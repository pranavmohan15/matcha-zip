# 🍵 Matcha Blanc

**Matcha Blanc** is a cinematic scrollytelling web experience showcasing an artisanal **matcha + espresso fusion drink**.
It uses a custom HTML5 Canvas animation synchronized to scroll to create a smooth frame-by-frame storytelling effect.

### 🌐 Live Demo

https://matcha-zip-tau.vercel.app/

---

## ✨ Features

* 196-frame **scroll-driven animation**
* Custom **HTML5 Canvas rendering engine**
* **Sticky typography storytelling**
* Premium **Dark Forest UI aesthetic**
* Smooth **Framer Motion transitions**
* Fully **responsive layout**

---

## 🛠 Tech Stack

* **Next.js 14**
* **React**
* **Tailwind CSS**
* **Framer Motion**
* **HTML5 Canvas**

---

## 🚀 Run Locally

```bash
git clone https://github.com/pranavmohan15/matcha-zip.git
cd mega-matcha-latte
npm install
npm run dev
```

Open **http://localhost:3000** in your browser.
